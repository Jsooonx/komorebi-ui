# References

This directory contains non-runtime project material that should not affect the main app build:

- `designs/`: exploratory or archived design projects
- `templates/`: rebrand/template source projects
- `media/`: source images and videos that are not served directly by the app
- `visual-reference/`: screenshots and visual inspiration material

The production app should continue to use runtime assets from `public/` and application code from `src/`.
