# Morrow

Morrow is a standalone Vite + React hero template for a strategic design studio. It recreates the reference composition with a centered navigation pill, left-aligned conversion copy, four layered portfolio cards, and a compact social-proof row.

## Assets

The four portfolio images in `public/assets/portfolio/` were created with the built-in image generation workflow for this template: Verdant Relay, Soft Signal, Night Index, and Tide Form.

## Development

```bash
npm install
npm run dev
npm run build
```
